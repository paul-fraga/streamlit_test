# 🪙 Interactive Probability Simulator

This is an interactive web application developed in Python. The project simulates flipping a coin multiple times and calculates the mean value of the results. As the number of attempts increases, the app plots the progress in real time, demonstrating how the calculated mean approaches its expected value (0.5).

**🔴 [View the Live Web App on Render](https://streamlit-test-g140.onrender.com)**
<br>
**🌐 [View the Interactive Case Study on my Portfolio](PON_AQUI_TU_LINK_DE_LOVABLE)**

### 🛠️ Technologies & Tools
* **Language:** Python
* **Web Framework:** Streamlit
* **Data Manipulation:** Pandas
* **Statistical Analysis:** SciPy
* **Cloud Deployment:** Render

### 💡 Key Features
* Intuitive graphical interface using widgets such as buttons and sliders.
* Dynamic, real-time chart updates.
* Experiment history storage using session state variables (`st.session_state`) to preserve values across continuous runs.

### 🚀 How to run this project locally

If you want to test the application in your own environment, follow these steps:

1. Clone this repository to your local machine.
2. Make sure you have the required dependencies listed in the `requirements.txt` file installed:
   ```bash
   pip install pandas scipy streamlit
   ```
3. Run the application from the command line in the root folder of the project:
   ```bash
   streamlit run app.py
   ```
4. Your browser will automatically open the local application.
